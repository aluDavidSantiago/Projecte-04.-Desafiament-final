# Clase Magistral: Estructura de Páginas Web

Antes de comenzar a diseñar la maqueta completa del e-commerce del cliente de **EverPia**, es fundamental entender los dos pilares esenciales de cualquier tienda online:  

- **Landing Page**  
- **Proceso de Checkout**  

Esta sesión combina los conceptos fundamentales de ambos elementos para ofrecer una visión completa del **recorrido del usuario**, desde que llega al sitio web hasta que finaliza la compra.

---

## Objetivos de la clase

1. Comprender la estructura de una **Landing Page** y su objetivo principal: captar la atención y guiar al usuario hacia la conversión.  
2. Analizar el **Proceso de Checkout**, sus pasos críticos y las mejores prácticas para minimizar abandonos.  
3. Entender la relación entre la Landing y el Checkout dentro de la experiencia del usuario.  
4. Aplicar estos conocimientos en la creación de prototipos y maquetas para el proyecto de EverPia.
